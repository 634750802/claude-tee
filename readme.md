## Usage

```shell
# Claude Code
code-tee \                                                                                                                  
    --stream-server-url "http://127.0.0.1:3000" \
    --stream-id "my-magic-stream-id" \
    --stream-message-id "my-magic-stream-message-id" \
    claude "Hi"

# Codex
code-tee \                                                                                                                  
    --target-url "http://127.0.0.1:3000" \
    --stream-id "my-magic-stream-id" \
    codex "Hi"
```
